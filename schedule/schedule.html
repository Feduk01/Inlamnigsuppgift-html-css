"Better Web" 2023
Conference Schedule

Immerse yourself in a world of cutting-edge
technologies and web development best practices

Three days of incredible immersion into current trends, innovations and perspectives in web development. Together we will explore the future of the industry and much more!

Day 1 (20/10)

Time
Seminar
09:00 - 10:30
Introduction to React Hooks - Explore the basics of React Hooks and how they can enhance your React applications.
11:00 - 12:30
CSS Grid Layout - Learn the basics and advanced techniques for creating responsive web layouts with CSS Grid.
14:00 - 15:30
Web Performance Optimization - Techniques and tools to measure and enhance the load times and responsiveness of web pages.
16:00 - 17:30
Web Accessibility - Best practices and tools for making your website accessible to all users.

Speaker of the day is

Sergey Brin

We are pleased to announce that Sergey Brin, co-founder of Google, will be joining us at our upcoming web development conference. Sergey, who not only pioneered Google's creation but also actively studied advanced web development technologies, will share his experience and expertise on the following topics

Time
Seminar
09:00 - 10:30
Vue.js: An Introduction - A walkthrough of the basics of Vue.js and its ecosystem.
11:00 - 12:30
Responsive Web Design with Flexbox - How to use Flexbox to create dynamic and responsive web layouts.
14:00 - 15:30
Web Animations with CSS and JS - Techniques for creating smooth transitions and animations on the web.

PWA: Building Progressive Web Apps - How to develop offline-ready, performance-optimized web apps.

Speaker of the day is

Tim Cook

As part of our upcoming Web Development Conference, we are pleased to introduce one of our keynote speakers, Tim Cook. Leading Apple and a pioneer in integrating technology into everyday life, Tim will share his thoughts on how innovation is shaping the web and its future.

Day 3 (22/10)

Time
Seminar
09:00 - 10:30
Svelte: A Fresh Approach to Frontend - Introduction to Svelte and its advantages over other frontend frameworks.
11:00 - 12:30
Web Security: Best Practices - Learn about the most common web security risks and how to best protect against them.
14:00 - 15:30

16:00 - 17:30
SEO Best Practices for Web Developers - Gain insights on optimizing your web content for better search engine visibility without compromising on user experience.

Speaker of the day is

Mark Zuckerberg

On the last day of our Web Development Conference, you'll be treated to an equally exciting keynote. Mark Zuckerberg, the visionary and soul of Facebook, will lead a number of sessions, including a deep dive into Svelte: A Fresh Approach to Frontend and current techniques in Web Security: Best Practices. The main focus of his presentation will be on the role of social media in today's web.
